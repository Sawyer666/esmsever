Проектирование

Разработка технической документации

           Специалисты «ЭСМ Север» имеют большой опыт в разработке технической документации любой сложности в следующих областях производства:
- разработка методик и программ проведения испытаний оборудования;
- разработка инструкций для эксплуатационного персонала;
- разработка технических отчетов по пуско-наладочным и режимно- наладочным испытаниям котельной, ИТП, тепловой сети;
- разработка проектно-сметной документации;
- разработка проекта предельно-допустимых выбросов веществ в атмосферный воздух;
- разработка проекта сокращения границ санитарно-защитной зоны;
- разработка проекта производства работы.
        «ЭСМ Север» по желанию Заказчика может взять на себя процесс согласования технической документации в надзорных органах.
